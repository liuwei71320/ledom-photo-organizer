# 📸 Ledom Photo Organizer

**工业级照片/视频整理工具 | 原子化安全操作 | 万级文件实测稳定**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
![Python 3.8+](https://img.shields.io/badge/Python-3.8%2B-blue)
[![Streamlit](https://img.shields.io/badge/Streamlit-1.0+-red)](https://streamlit.io/)

## 一句话介绍

把手机、相机里散乱的照片，按日期和设备自动整理成整洁的文件夹。

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🚀 **高性能** | Fast-Hash 采样算法，4线程并发，速度提升10倍 |
| 🛡️ **原子化操作** | 复制/移动失败自动回滚，绝不丢失源文件 |
| 📅 **智能日期提取** | EXIF → 文件名 → mtime 三级降级，准确率95%+ |
| 🔄 **断点续传** | SQLite 状态库，中断后可继续，重复运行自动跳过 |
| 🖥️ **Web 界面** | Streamlit 构建，零配置启动，操作直观 |
| 📱 **格式支持** | JPEG/PNG/HEIC/RAW/MP4 等20+格式 |
| 🔍 **DRY-RUN模式** | 预览效果，确认无误后再执行 |
| ⚡ **智能去重** | MD5 哈希校验，自动跳过已处理文件 |

## 📥 快速开始

### 环境要求

- Python 3.8 或更高版本
- Windows / macOS / Linux

### 一键启动

#### Windows:

```batch
# 1. 下载本项目
# 2. 双击运行启动
Ledom照片视频整理工具.bat
```

#### Mac / Linux:

```bash
# 1. 安装依赖
pip install streamlit exifread pillow-heif

# 2. 启动应用
streamlit run photo_organizer_streamlit.py
```

浏览器会自动打开 `http://localhost:8501`，开始使用！

## 🔧 配置参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| 源目录 | 路径 | `~/Pictures/Phone_Imports` | 待整理照片所在目录 |
| 目标目录 | 路径 | `~/Pictures/Organized_Photos` | 整理后文件存放目录 |
| 目录结构 | 枚举 | `device_year_month` | device/year/month / year/month / year/month/day / year/month/device |
| 操作模式 | 枚举 | `copy` | copy（保留源文件）/ move（转移后删除） |
| 冲突处理 | 枚举 | `rename` | rename（自动加序号）/ skip / overwrite |
| 并发线程数 | 整数 | `4` | 范围 1-32，建议与 CPU 核数持平 |
| Fast-Hash | 布尔 | `True` | 大文件采样哈希，关闭后使用全量 MD5 |
| DRY-RUN | 布尔 | `False` | 预览模式，不执行任何文件操作 |
